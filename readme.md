# Deep Learning

Repositorio de la materia **Deep Learning** de la Especialización en Inteligencia Artificial.

**Estudiante:** Wilson Alfonso Diaz Capador

Este repositorio contiene los materiales, ejercicios y entregas correspondientes a la clase de Deep Learning. Cada carpeta `weekN` corresponde a una unidad o semana del curso.

## Estructura del repositorio

```
deep_learning/
├── readme.md              # Este archivo (visión general de la clase)
├── week1/                  # Week 1 - Neurona vainilla
│   ├── README.md
│   └── ejercicio_neurona_vainilla.ipynb
├── week2/                  # Week 2 - Redes neuronales básicas
│   ├── README.md
│   └── ejercicio1.py
└── ...
```

## Contenido por week

- **Week 1:** Ejercicio neurona en 15 minutos (red vainilla, bias, comportamiento AND). Ver [week1/README.md](week1/README.md).
- **Week 2:** Implementación de redes neuronales básicas (Perceptrón, red una capa, red multicapa). Ver [week2/README.md](week2/README.md). Aun en desarrollo

## Cómo usar este repositorio

Los notebooks y scripts están preparados para ejecutarse en **Google Colab** o en un entorno local con Python 3 y las dependencias indicadas en cada entrega (por ejemplo, NumPy y Matplotlib en la semana 1).
